# Frontend Project

React + Next.js + Tailwind CSS 프로젝트입니다.

## 시작하기

### 의존성 설치

```bash
npm install
```

또는

```bash
yarn install
```

또는

```bash
pnpm install
```

### 개발 서버 실행

```bash
npm run dev
```

또는

```bash
yarn dev
```

또는

```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

### 프로덕션 빌드

```bash
npm run build
npm run start
```

## 프로젝트 구조

- `app/` - Next.js App Router 디렉토리
- `components/` - React 컴포넌트
- `public/` - 정적 파일
- `tailwind.config.ts` - Tailwind CSS 설정
- `tsconfig.json` - TypeScript 설정

## 기술 스택

- React 18
- Next.js 14
- TypeScript
- Tailwind CSS
